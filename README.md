## Propuesta
Mi librero es mi lugar personal, mi pequeño espacio, así que lo tomé como inspiración para crear la aplicación web "Mi librero". Será una web en la cual se podrá encontrar información acerca de libros y autores, de los cuales he tenido la oportunidad de leer. Me inspiré en la aplicación Goodreads para añadir la opción de ingreso y registro de usuario, como también la funcionalidad de dar una reseña sobre los libros o una opinión sobre los autores, las cuales serán mostradas tipo foro. La aplicación no tendrá la opción para subir información de un nuevo libro o autor, ésto se deberá realizar, por el momento, desde el código.

### ¿Qué se podrá visualizar?
- Información acerca de un libro: nombre del libro, nombre del autor, género, fecha de publicación, puntuación por el librero, puntuación promedio, estadística obtenida a partir de las calificaciones dadas por los usuarios, sinopsis del libro, reseña escrita por mí.
- Información acerda de un autor: nombre del autor, lugar de nacimiento, fecha de nacimiento, fecha de fallecimiento, género predominante en sus obras, puntuación promedio por el librero, puntuación promedio de sus obras, estadística obtenida a partir de las puntuaciones dadas a sus diferentes libros, biografía corta, opinión sobre el autor.
- Encontrar los libros divididos según su género.

### Funcionalidades
Se podrá:
- Registrar usuarios que luego podrán ingresar a la web.
- Cambiar la contraseña de un usuario en caso de olvido.
- Publicar reseñas sobre libros y opiniones sobre autores (cabe señalar que todo lo publicado será de manera anónima).
