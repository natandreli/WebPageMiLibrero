## Propuesta
Mi librero es mi lugar personal, mi pequeño espacio, así que lo tomé como inspiración para crear la aplicación web "Mi librero". Será una web en la cual se podrá encontrar información acerca de libros y autores, de los cuales he tenido la oportunidad de leer. Me inspiré en la aplicación Goodreads para añadir la opción de ingreso y registro de usuario, como también la funcionalidad de dar una reseña sobre los libros o una opinión sobre los autores, las cuales serán mostradas tipo foro. La aplicación no tendrá la opción para subir información de un nuevo libro o autor, ésto se deberá realizar, por el momento, desde el código.
